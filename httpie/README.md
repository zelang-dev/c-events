# httpie - a dual webclient/server processor

This *subproject* a **WIP** implementation that follows up on [LibHTTP](https://github.com/lammertb/libhttp), and [CivetWeb](https://github.com/civetweb/civetweb), forks of [Mongoose](https://github.com/cesanta/mongoose).

- Where as, all *parsing* and *actual communcations* replaced with **Events API**,
and other routines that's already implemeneted.
